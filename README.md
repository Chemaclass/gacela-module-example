# Gacela Module example

This is an example of a module using [Gacela Framework](http://gacela-project.com/).

### Example Gacela structure

```bash
application-name
├── gacela.php
├── config
│   └── default.php
│
├── src
│   └── Customer
│       ├── Domain
│       │   └── Generator
│       │       ├── NicknameGenerator.php
│       │       └── NicknameGeneratorInterface.php
│       ├── CustomerConfig.php
│       ├── CustomerDependencyProvider.php
│       ├── CustomerFacade.php
│       ├── CustomerFacadeInterface.php
│       └── CustomerFactory.php
│
├── tests
│   └── ...
└── vendor
    └── ...
```
